## Agent skills

### Issue tracker

Issues and PRDs live as GitHub issues, managed via the `gh` CLI. See `agent-docs/config/issue-tracker.md`.

### Triage labels

Five canonical triage roles, each mapped 1:1 to its default label string. See `agent-docs/config/triage-labels.md`.

### Domain docs

Single-context: `agent-docs/CONTEXT.md` + `agent-docs/adr/`. See `agent-docs/config/domain.md`.
